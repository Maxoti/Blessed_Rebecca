# Blessed Rebecca School Website

A simple, clean, responsive static website for Blessed Rebecca School.

## Structure
- index.html – Home page
- about.html – About the school
- admissions.html – Admissions info
- contact.html – Contact page
- assets/css/style.css – Global styling
- assets/js/main.js – Scripts
- assets/images – Images used in the project

## Technologies
- HTML5
- CSS3
- JavaScript (optional)

## Usage
Open `index.html` in a browser to view the website.
